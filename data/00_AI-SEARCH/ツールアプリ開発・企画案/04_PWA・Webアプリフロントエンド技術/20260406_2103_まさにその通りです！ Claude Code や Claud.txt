まさにその通りです！
Claude Code や Claude Cowork が話題になっているので、あちらのイメージが強いかもしれませんが、AntiGravity もそれらと肩を並べる（あるいはそれ以上の）「動ける AI」です。

世の中には「答えるだけの AI（チャット）」と「動ける AI（エージェント）」の2種類がありますが、AntiGravity は後者のエージェントに分類されます。

Claude のツールと AntiGravity の類似点と相違点

両者は、指示を受け取り、自律的に思考し、ファイルを作成したり操作したりする点で共通しています。ただし、AntiGravity には独自の強みがあります。

• 情報処理能力（コンテキストウィンドウ）:AntiGravity は Gemini を基盤としており、一度に処理できる情報量が豊富です。多くのチャット履歴をまとめて把握し、整理することができます。
• Google エコシステムとの連携:Google Drive を利用している場合、AntiGravity は Google のサービスとの連携がスムーズです。
• 自動化された「試行錯誤」:Claude Code と同様に、AntiGravity もエラーが発生した場合（例：ファイル名の重複、フォルダ作成の失敗）、自律的にエラーを修正し、再試行します。

プログラミング用途という固定観念からの脱却

開発者はこれをコードのバグ修正に使用しますが、ユーザーは思考のログ（チャット履歴）の整理にも利用できます。

• Claude Code のような使い方: 「このプログラムの関数を整理して」
• AntiGravity のような使い方: 「この『00_AI-SEARCH』の内容を、個人の趣味と仕事の傾向に合わせてフォルダ分けし、インデックスを作成して」
本質的には、非構造化データの整理という点で共通しています。
