# 指示
あなたは優秀なフロントエンドエンジニアです。
以下の企画内容と要件に基づいて、ブラウザ上で動作する「インラインベクター動画制作ツール（簡易版）」の設計および実装コード（HTML, CSS, Vanilla JavaScript）を作成してください。

# ツール概要
PWAツールの紹介ページに埋め込むための「超軽量・簡素で、ページに溶け込む音声付き解説アニメーション」を作成するエディタツールです。
動画ファイル（MP4等）を使用せず、「文字や図形のデータ（JSON）」と「音声ファイル」をJavaScriptで完全に同期させて再生する仕組みを構築するためのデータ書き出しを行います。

# コア機能と要件
1. 素材の配置とプレビュー
・「ブラウザの枠」「赤い矢印」「指」などのSVGパーツを画面上に配置できる。
・背景は透明として扱う。

2. 音声ファイルの読み込みと同期（タイムライン機能）
・ユーザーが用意した音声ファイル（mp3, aac, webm等）をアップロード（読み込み）できる。
・音声のタイムコードに合わせて、パーツのアクション（例: 「0秒: 矢印を点滅」「2秒: 指アイコンを指定座標へ移動」など）を設定できる超シンプルなUI。

3. エクスポート（出力）機能
・配置したSVGパーツの情報や、タイムラインのアクション情報を含んだ「アニメーションデータ（JSON）」を出力できる。
・最終的に、ウェブサイトに埋め込むための「再生用プレイヤーの基礎コード（JSONと音声を読み込んで同期再生するJavaScript）」も合わせて提供する。

# 技術スタック
・HTML5 / CSS3
・Vanilla JavaScript (フレームワークなし)
・Web Audio API (音声制御用)
・Web Animations API または CSS アニメーション (SVGの動作制御用)

# 設計上の注意点
・圧倒的な軽さと解像度非依存（ボケないこと）を担保するため、描画はすべてDOM/SVGで行うこと。
・AIや外部APIなどの高度で複雑な処理には依存せず、現代のWeb標準技術のみで構築すること。
・開発者の保守性を高めるため、コードはシンプルにモジュール化（または関数ごとに役割を明確化）すること。

2. ツール本体の画面UIとなるHTML/CSS
3. プレビューおよびJSON出力処理を担うJavaScriptコード
4. 出力されたJSONと音声を、実際の紹介ページで再生するための「軽量プレイヤー用JSスクリプト」の雛形
