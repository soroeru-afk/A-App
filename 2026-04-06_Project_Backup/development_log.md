## A-Clock 開発ログ

### プロジェクト概要
- **目的**: ミニマルなデジタル時計の作成
- **ファイル構成**:
  - `index.html` — HTML構造
  - `style.css` — スタイル
  - `script.js` — 動作ロジック

### デザイン方針（カオルさまの好み）
- 深い黒背景（`#050505` / 中央から `#151515` へのラジアルグラデーション）
- 白〜ライトグレーのセグメント風テキスト（`#e8e8e8`）
- 柔らかいグロー（text-shadow）
- ぼかした反射（リフレクション）エフェクト
- フォント: `Share Tech Mono`（Google Fonts）
- 大人っぽく洗練された雰囲気

### 実装済み機能
- [x] 現在時刻のリアルタイム表示（HH:MM:SS、200msごとに更新）
- [x] 深い黒背景 + ラジアルグラデーション
- [x] グロー効果（text-shadow）
- [x] 下方向への反射エフェクト（ぼかし＋フェードアウト）

### 現在のファイル状態（2026-04-01 時点）

**index.html** — シンプルな2div構成（`.clock` + `.reflection`）

**style.css** — `Share Tech Mono`、`clamp(4rem, 10vw, 12rem)` のレスポンシブフォント、反射は `scaleY(-1)` + `blur(6px)` + マスクグラデーション

**script.js** — `setInterval(updateClock, 200)`、DOM更新は変化時のみ
